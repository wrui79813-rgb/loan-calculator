# 💰 私人借贷计算器

纯前端的多人多笔借贷管理工具，无需后台，数据保存在浏览器本地。

## 功能特性

1. ✅ **多人多笔借款**：按借款人分组，每笔独立管理
2. ✅ **5种还款方式**：先息后本、先本后息、等额本金、等额本息、到期一次性还本付息
3. ✅ **4种还款模式**：自动抵扣、只还息、只还本、手动自由拆分
4. ✅ **单利/复利切换**：一键切换，自定义逾期罚息利率
5. ✅ **计算过程可视化**：每笔利息天数、公式、抵扣逻辑全部展示
6. ✅ **导出功能**：Excel 下载、TXT 对账文件、一键复制
7. ✅ **数据本地保存**：localStorage 持久化，刷新不丢
8. ✅ **撤销还款**：任意一笔可撤销，账务自动回滚
9. ✅ **简洁界面**：大字体、大按钮、老年人友好
10. ✅ **纯离线**：无后台、无数据收集、无网络依赖

## 计算逻辑

- **日利率** = 年利率 ÷ 365
- **单利** = 本金 × 日利率 × 天数
- **复利** = 本金 × (1 + 日利率)^天数 - 本金
- **自动抵扣顺序**：先扣利息，余额扣本金
- **逾期罚息**：超过到期日自动切换罚息利率

### 验算示例

10000元本金，年利率12%，借30天，单利：
- 10000 × 0.12 / 365 × 30 = **98.63元** ✓
